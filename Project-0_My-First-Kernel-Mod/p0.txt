.SH NAME
Natalie Poche \- P0: My First Kernel Mod

.SH SYNOPSIS
Modify the Reptilian kernel boot sequence to display a custom log message and personal identifier. Generate a unified patch file and personalize the GRUB menu for extra credit.

.SH DESCRIPTION
This project involves modifying the Reptilian OS kernel to print a specific string during the boot process.

.SS /usr/rep/src/reptilian-kernel/init/main.c
.TP
.B Identification
Located via the requirement to find the 
.I rcu_end_inkernel_boot()
function.
.TP
.B Changes
Inserted a 
.I printk
statement: printk(KERN_ERR "##### Natalie Poche (UFID: 1533-9442) Go Gators! #####"); with newlines inserted before and after the log message.
.TP
.B Logic
Placed to execute just before "Detecting Reptilian system partition" to ensure visibility during default boot. Used KERN_ERR to ensure minimum level for log to display during boot.

.SS /mnt/sysroot/grub/menu.lst (Extra Credit)
.TP
.B Identification
Directly edited the configuration file using a text editor as mandated.
.TP
.B Changes
Appended "(Natalie Poche)" to the default "Reptilian 20.08-A9.0-r2" menu entry.
.TP
.B Logic
Personalizes the GRUB boot selection screen without using restricted GRUB utilities. Directory /mnt/sysroot/grub is android part that contains menu.lst [Cite: 2].
